Index

Identifi index root. Contains five indexes: identitiesBySearchKey, identitiesByTrustDistance, messagesByHash, messagesByTimestamp, messagesByDistance. If you want messages saved to IPFS, pass options.ipfs = instance.

When you use someone else's index, initialise it using the Index constructor

new Index(gun: Object, options: Object): Index
Parameters
gun (Object) gun node that contains an Identifi index (e.g. user.get('identifi'))
options (Object) see default options in example
Returns
Index: Identifi index object
Example
Default options:
{
 ipfs: undefined,
 indexSync: {
   importOnAdd: {
     enabled: true,
     maxMsgCount: 500,
     maxMsgDistance: 2
   },
   subscribe: {
     enabled: true,
     maxMsgDistance: 1
   },
   query: {
     enabled: true
   },
   msgTypes: {
     all: false,
     rating: true,
     verification: true,
     unverification: true
   }
 }
}

Message

Identifi message: an object that has an author, recipient, signer, type, timestamp, context and optionally other fields.

On Identifi, signer and author can be different entities. This enables the crawling of content from existing datasets. That makes Identifi an useful search tool even with no initial userbase.

Messages are serialized as JSON Web Signatures.
